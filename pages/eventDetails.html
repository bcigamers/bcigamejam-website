Event Details

The BCI Game Jam series is a premier game development event focused around building new interactive experiences designed specifically for use in brain-computer interfaces (BCIs). Run annually by BCI Games, this event leverages expert knowledge of cutting-edge BCI technology with the creative minds of game development communities around the globe. Our mission is to build brand new ways to play and create games that are accessible to everyone.
